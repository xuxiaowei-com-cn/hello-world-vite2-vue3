# hello-world-vite2-vue3
 
## 创建项目
- 创建完成功，项目路径：create-app/hello-world-vite2-vue3
- [Vue 3 Documentation](https://v3.vuejs.org)
~~~
mkdir create-app
cd create-app
npm i @vitejs/create-app
npm init @vitejs/app hello-world-vite2-vue3 --template vue
~~~
